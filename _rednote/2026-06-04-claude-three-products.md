# 你可能只用了 Claude 三分之一的能力 👀

很多人认识 Claude，是从打开 claude.ai 问问题开始的。

但其实 Anthropic 做了三个产品——**Chat、Code、Cowork**，三种完全不同的使用方式，能做的事差别很大。

简单说一下各自是什么 👇

---

## 🗨️ Chat：最熟悉的那个

就是网页版 / App 里的对话模式。

问问题、写文章、分析资料、头脑风暴……来回对话，随开随用。付费用户还可以上传资料，让它直接生成 PPT、Excel、Word，做好给你下载链接。

**适合：** 日常使用、内容创作、资料整理

---

## 💻 Code：真正能帮你把事"做完"的那个

很多人以为 Claude Code 是程序员专属，其实不是。

它和 Chat 最大的区别是——**它能直接访问你的本地文件夹，帮你真正把东西生成出来**。写论文、做 PPT、整理表格、生成报告……不只是给你建议，而是直接把文件做好放在那里。

改了什么、写了什么，全程可见，随时介入。

**适合：** 任何需要批量生成、处理本地文件的场景，不只是写代码

---

## 🤝 Cowork：Code 的能力，普通人也能用

Cowork 内置在桌面 App 里，本质上用的是和 Code 一样的底层能力——**区别只是把技术细节藏起来了**，不需要你做任何技术操作，打开桌面应用就能直接用。

同样能访问本地文件夹，同样能帮你生成文档、整理资料，但多了几个 Code 没有的功能：

✅ **委托后可以走开**——把任务交出去就去忙别的，它后台跑完再通知你

✅ **支持定时任务**——设置每周自动整理文件夹、生成周报，不用每次手动触发

✅ **内置丰富插件**——销售、财务、HR、市场……装上即用

代码运行在隔离沙盒里，永久删除文件这类操作必须你明确授权才会执行，安全有保障。

> ⚠️ 需要 Pro 及以上付费计划，任务运行期间桌面 App 需要保持开着

**适合：** 不想做任何技术操作、但想用上 Code 完整能力的普通用户

---

如果你平时只用 Chat，其实已经够用了。但如果你想让 Claude 真正碰到你的本地文件、直接把文档做出来，Code 和 Cowork 都能做到。
