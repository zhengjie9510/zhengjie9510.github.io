# 你可能只用了 Claude 三分之一的能力

很多人认识 Claude，是从 claude.ai 开始的——打开网页，输入问题，它回答。但 Anthropic 其实做了三个产品：**Chat、Code、Cowork**。同一个 AI，三种完全不同的使用方式。

---

## 01 Chat：对话是核心，也能帮你做东西

Chat 就是大家最熟悉的那个——claude.ai 网页版、手机 App、桌面 App 里的 Chat 模式，都是它。

它的核心是对话。问问题、写文章、分析资料、讨论想法，这是它最擅长的场景。你说一句，它回一句，来回推进。

付费用户还可以让它直接生成文件。你把资料上传给它，描述你要什么，它可以帮你做出 PPT、Excel、Word 文档，做好了给你下载链接。不过它只能处理你上传的内容，看不到你电脑本地的文件夹。

随开随用，不需要安装，免费账号就能开始。

---

## 02 Code：开发者的编程搭档

Claude Code 是专门为开发者设计的工具，在终端里运行，也有 VS Code 和 JetBrains 的插件版本。

它和 Chat 最大的不同是：它能直接在你的**本地环境**里干活。读你的文件、写代码、改文件、执行命令、调试报错，整个项目文件夹都在它的访问范围内。它改了什么、写了什么，你全程可以看到，也可以随时介入。

对开发者来说，这相当于一个随时待命的编程搭档，能接手复杂的工程任务，从头跑到尾，大幅减少重复性的工作。

---

## 03 Cowork：把 Code 的能力带给普通人

Claude Code 很强，但需要打开终端、懂命令行，对普通用户门槛太高。所以 Anthropic 做了 Cowork，内置在 Claude 桌面 App 里。

官方的定位很直接：**把 Claude Code 的 agent 能力，用不需要任何技术背景的方式，带给普通用户。**

和 Code 一样，Cowork 也能直接访问你授权的本地文件夹，读取内容、生成文件、整理和修改文档，完成复杂的多步骤任务。对于非开发者，它还有几个 Code 没有的优势：

**🔹 委托后可以走开。** Chat 需要你全程在旁边一步步推进，Cowork 不一样——把任务交出去就可以去做别的事，它在后台跑完再通知你。

**🔹 支持定时任务。** 可以设置让它定期自动执行，比如每周自动整理一次文件夹、生成一份周报，不需要每次手动触发。

**🔹 内置丰富的插件。** 覆盖销售、财务、法务、市场、HR、设计等常见工作场景，装上即用，无需自己配置。

安全方面，Cowork 的代码运行在你电脑上一个隔离的沙盒里，不会影响系统其他部分，永久删除文件这类操作也必须经过你明确授权才会执行。

> ⚠️ 需要注意：Cowork 需要 Pro 及以上的付费计划，任务运行期间桌面 App 需要保持开着。

---

## 一张表看清三者区别

| | Chat | Code | Cowork |
|---|---|---|---|
| 主要定位 | 对话助手 | 开发者编程搭档 | 普通用户任务执行器 |
| 访问本地文件 | ❌ | ✅ | ✅ |
| 需要技术背景 | 否 | 是 | 否 |
| 委托后可以走开 | ❌ | ✅ | ✅ |
| 支持定时任务 | ❌ | ❌ | ✅ |
| 使用方式 | 网页 / App | 终端 / IDE 插件 | 桌面 App |

---

如果你平时只用 Chat，它已经能覆盖大多数日常需求。但如果你想让 Claude 真正接触到你本地的文件、自主完成复杂任务，Cowork 就是为你准备的——不需要懂技术，下载桌面 App 就能开始。
